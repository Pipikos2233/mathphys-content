## §1.1. Действительные числа. Числовая ось
- Рациональные и иррациональные числа; множество действительных чисел R.
- Изображение действительных чисел точками числовой оси; взаимно однозначное
  соответствие между R и точками оси.
- Всякое иррациональное число приближается рациональными с любой точностью
  (десятичные приближения по недостатку и по избытку; формулировки давай для
  положительных чисел, случай отрицательных не разбирай).

## §1.2. Абсолютная величина
- Определение |x| (через два случая).
- Свойства: |x| ≥ 0; |−x| = |x|; неравенство |x + y| ≤ |x| + |y|;
  |x − y| ≥ |x| − |y|; |xy| = |x||y|; |x/y| = |x|/|y| (y ≠ 0).
- Геометрический смысл |x − a| как расстояния; запись окрестности
  |x − a| < δ (всюду δ > 0).

## §1.3–§1.5. Переменные и постоянные величины
- Переменная и постоянная величина; область изменения переменной.
- Промежутки: интервал, отрезок, полуинтервал, бесконечные промежутки;
  окрестность точки, δ-окрестность.
- Упорядоченная переменная величина; возрастающая, убывающая, монотонная
  переменная; ограниченная переменная.

## §1.6. Функция
- Определение функции одной переменной (каждому x из области определения
  соответствует одно значение y); аргумент, область определения, область
  значений.
- Обозначения y = f(x); значение функции в точке.
- Чётные и нечётные функции (определения; область определения предполагай
  симметричной относительно нуля и указывай это условие явно).
- Периодические функции: определение давай в стандартном полном виде —
  T ≠ 0 есть период f, если для всякого x из области определения точки
  x + T и x − T тоже принадлежат ей и f(x + T) = f(x); наименьший
  положительный период (основной) существует не всегда (пример: постоянная
  функция на R).

## §1.7. Способы задания функции
- Табличный, графический, аналитический способы; график функции.
- Функции, заданные несколькими формулами на разных участках.
- Естественная область определения аналитически заданной функции (примеры:
  корень чётной степени, дробь, логарифм).

## §1.8. Основные элементарные функции. Элементарные функции
- Перечень основных элементарных функций и их свойства СЛОВАМИ, без графиков.
  Обязательный набор свойств для каждой функции указан ниже; ничего сверх
  этого набора не требуется:
  степенная y = x^a — только случаи a = 2 (область определения R, чётная,
  убывает на (−∞, 0], возрастает на [0, +∞)) и a = 3 (область определения R,
  нечётная, возрастает на R); для прочих a достаточно замечания, что область
  определения зависит от показателя, с примерами x^(1/2) (x ≥ 0) и x^(−1)
  (x ≠ 0);
  показательная y = a^x (a > 0, a ≠ 1) — область определения, область
  значений, монотонность в зависимости от a;
  логарифмическая y = log_a x (a > 0, a ≠ 1) — область определения, область
  значений, монотонность в зависимости от a;
  тригонометрические sin, cos, tg, ctg — только области определения,
  области значений, чётность/нечётность и периоды (промежутки монотонности
  НЕ требуются);
  обратные тригонометрические arcsin, arccos, arctg, arcctg — только
  области определения и области значений (монотонность и чётность
  НЕ требуются).
- Определение элементарной функции: конечная цепочка арифметических операций
  и суперпозиций над основными элементарными функциями.

## §1.9. Алгебраические функции
- Целая рациональная функция (многочлен), дробно-рациональная функция;
  иррациональные функции описывай примерами (√x, x^(2/3) и т.п.), без попытки
  строгого синтаксического определения; алгебраические и трансцендентные
  функции (без доказательства трансцендентности).

## §1.10. Полярная система координат
- Полюс, полярная ось; полярные координаты ρ ≥ 0, φ.
- Связь с декартовыми: x = ρ cos φ, y = ρ sin φ;
  ρ = √(x² + y²); угол φ определяй по знакам x и y (для x ≠ 0 через
  tg φ = y/x с выбором четверти, для x = 0, y ≠ 0 это ±π/2; в полюсе угол
  не определён).
- Примеры кривых в полярных координатах: окружность ρ = a (a > 0),
  спираль Архимеда ρ = aφ (a > 0, φ ≥ 0).

## Ориентиры для теста
- Уметь: находить естественную область определения; проверять чётность/
  нечётность; находить период; вычислять значения композиций; переводить
  точки между декартовыми и полярными координатами; решать неравенства
  с модулем вида |x − a| < δ.
- Числовые задачи должны иметь точный ответ, проверяемый в SymPy.
