## §2.1. Предел переменной величины
- Определение предела упорядоченной переменной величины (числовой
  последовательности): a — предел x_n, если для всякого ε > 0 найдётся
  номер N (натуральное), такой что |x_n − a| < ε при всех n > N.
- Геометрический смысл: вне любой ε-окрестности точки a лежит лишь конечное
  число членов.
- Предел постоянной равен ей самой; переменная не может иметь двух разных
  пределов (формулировка без доказательства).
- Бесконечно большая переменная величина: x_n → ∞, если для всякого M > 0
  найдётся N, такое что |x_n| > M при n > N. Подчеркни: «предел равен
  бесконечности» — условное выражение, предела (числа) у неё нет.
- Ограниченная переменная (определение). Сходящаяся (имеющая конечный предел)
  переменная ограничена (без доказательства); обратное неверно — пример
  x_n = (−1)^n.

## §2.2. Предел функции
- Определение предела функции при x → a (по Коши, на языке ε–δ): число b —
  предел f(x) при x → a, если f определена в некоторой окрестности точки a
  (кроме, быть может, самой точки a) и для всякого ε > 0 существует δ > 0,
  такое что |f(x) − b| < ε при всех x с 0 < |x − a| < δ.
  Подчеркни условие 0 < |x − a| (значение f(a) не участвует).
- Односторонние пределы (слева и справа): определения; предел существует
  тогда и только тогда, когда оба односторонних существуют и равны.
- Предел при x → ∞ (и x → ±∞): определение через «для всякого ε > 0
  существует M > 0, такое что |f(x) − b| < ε при |x| > M».

## §2.3. Функция, стремящаяся к бесконечности. Ограниченные функции
- f(x) → ∞ при x → a: определение через M и δ; варианты +∞ и −∞.
- Ограниченная функция на множестве; функция, ограниченная при x → a
  (в некоторой окрестности) и при x → ∞.
- Функция, имеющая конечный предел при x → a, ограничена в некоторой
  проколотой окрестности a (без доказательства).
- 1/f: если f(x) → ∞, то 1/f(x) → 0; если f(x) → 0 и f(x) ≠ 0 в проколотой
  окрестности, то 1/f(x) → ∞ (формулировки с условиями).

## §2.4. Бесконечно малые и их основные свойства
- Определение: α(x) — бесконечно малая при x → a (или x → ∞), если её предел
  равен 0.
- Связь с пределом: f(x) → b тогда и только тогда, когда f(x) = b + α(x),
  где α — бесконечно малая.
- Свойства (формулировки; все — при одном и том же стремлении аргумента):
  сумма КОНЕЧНОГО числа бесконечно малых — бесконечно малая;
  произведение бесконечно малой на ограниченную функцию — бесконечно малая
  (следствия: произведение бесконечно малых, произведение постоянной на
  бесконечно малую);
  частное α(x)/g(x) — бесконечно малая, если g(x) имеет ненулевой предел.

## §2.5. Основные теоремы о пределах
- Теоремы об арифметике пределов: предел суммы, произведения, частного
  (для частного — предел знаменателя отличен от нуля). Все — в предположении,
  что оба предела существуют и конечны; конечное число слагаемых/сомножителей.
- Предельный переход в неравенстве: если f(x) ≤ g(x) в проколотой окрестности
  и оба предела существуют, то lim f ≤ lim g (строгое неравенство может
  превратиться в нестрогое — отметь).
- Теорема о двух милиционерах (о промежуточной функции): формулировка со
  всеми условиями.
- Монотонная ограниченная переменная имеет предел (формулировка, без
  доказательства).

## §2.6. Первый замечательный предел
- lim (sin x)/x = 1 при x → 0 (x в радианах — подчеркни); идея доказательства
  через сравнение площадей/дуг НЕ требуется, достаточно формулировки.
- Следствия: lim (tg x)/x = 1, lim (sin kx)/x = k при x → 0 (k ≠ 0),
  lim (1 − cos x)/x² = 1/2.

## §2.7. Число e
- Определение: e = lim (1 + 1/n)^n при n → ∞ (существование — по теореме
  о монотонной ограниченной переменной, без доказательства); 2 < e < 3,
  e ≈ 2,71828 — иррационально.
- Второй замечательный предел: lim (1 + 1/x)^x = e при x → ±∞;
  эквивалентная форма lim (1 + α)^{1/α} = e при α → 0.

## §2.8. Натуральные логарифмы
- ln x = log_e x; связь с десятичным логарифмом: lg x = M ln x,
  где M = lg e ≈ 0,4343; ln x определён при x > 0.

## §2.9. Непрерывность функций
- Определение непрерывности f в точке a: f определена в некоторой окрестности
  точки a (включая саму точку) и lim f(x) при x → a равен f(a).
- Эквивалентная формулировка через приращения: Δy → 0 при Δx → 0.
- Односторонняя непрерывность; непрерывность на отрезке [a, b]
  (на концах — односторонняя).
- Точки разрыва: определение; классификация — разрыв первого рода (оба
  односторонних предела конечны; в т.ч. устранимый разрыв, когда они равны
  между собой, но не равны f(a) или f(a) не определено) и второго рода
  (хотя бы один односторонний предел не существует или бесконечен).
  Примеры: y = 1/x в нуле (второго рода), y = sgn x в нуле (первого рода).
- Непрерывность суммы, произведения, частного (знаменатель ≠ 0 в точке),
  сложной функции; непрерывность основных элементарных функций во всех
  точках их областей определения (формулировки без доказательств).

## §2.10. Некоторые свойства непрерывных функций
- Все теоремы формулируй для функции, непрерывной НА ОТРЕЗКЕ [a, b] —
  замкнутость отрезка обязательна, для интервала теоремы неверны
  (приведи контрпример: y = 1/x на (0, 1) не ограничена).
- Первая теорема Вейерштрасса: непрерывная на отрезке функция ограничена
  на нём.
- Вторая теорема Вейерштрасса: непрерывная на отрезке функция достигает
  на нём своих наибольшего и наименьшего значений.
- Теорема Больцано–Коши о нуле: если f непрерывна на [a, b] и f(a)·f(b) < 0,
  то существует точка c ∈ (a, b), где f(c) = 0.
- Теорема о промежуточных значениях: непрерывная на [a, b] функция принимает
  все значения между f(a) и f(b).

## §2.11. Сравнение бесконечно малых
- Пусть α, β — бесконечно малые при одном стремлении аргумента и β ≠ 0
  в проколотой окрестности. Определения через предел отношения α/β:
  одного порядка (конечный ненулевой предел), высшего порядка (предел 0,
  запись α = o(β)), эквивалентные (предел 1, запись α ~ β).
- Таблица эквивалентностей при x → 0: sin x ~ x, tg x ~ x, arcsin x ~ x,
  arctg x ~ x, 1 − cos x ~ x²/2, ln(1 + x) ~ x, e^x − 1 ~ x,
  (1 + x)^k − 1 ~ kx (k ≠ 0).
- Замена на эквивалентную бесконечно малую допустима в пределах отношения
  и произведения; в суммах и разностях — НЕЛЬЗЯ (контрпример:
  lim (tg x − sin x)/x³ при x → 0 равен 1/2, а «замена» tg x ~ x, sin x ~ x
  дала бы 0).

## Ориентиры для теста
- Уметь: вычислять пределы через арифметику пределов и замечательные пределы;
  раскрывать простейшие неопределённости 0/0 с многочленами (сокращение
  множителя) и с sin/tg; применять эквивалентности; определять тип точки
  разрыва; проверять применимость теорем §2.10 (отрезок vs интервал).
- Числовые задачи должны иметь точный ответ, проверяемый в SymPy (limit).
- Обязательно ≥1 вопрос на условия применимости (замкнутость отрезка в
  теоремах о непрерывных функциях или запрет замены эквивалентных в разности).
